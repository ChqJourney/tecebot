# Tecebot 产品官网

认证检测行业 AI Agent 产品官方网站。

## 🌐 在线访问

**GitHub Pages**: https://patrickstar-shang.github.io/tecebot-website/

## ✨ 产品特色

- **标准智能解读** - 深度学习检测标准，自动解析复杂条款
- **辅助智能判定** - 基于数据与标准要求，智能辅助合格判定
- **智能数据提取** - 从检测文档中自动提取关键信息
- **报告智能生成** - 自动生成格式规范、内容完整的检测报告
- **报告智能审核** - 对检测报告进行全方位智能审核

## 🎨 设计风格

- **智慧** - 现代科技感，深蓝色主调
- **精准** - 清晰的信息架构，专业布局
- **可定制** - 灵活的配置能力，深度适配

## 🛠️ 技术栈

- HTML5
- CSS3 (CSS Variables, Flexbox, Grid)
- Vanilla JavaScript (ES6+)
- GitHub Pages 部署

## 📁 项目结构

```
tecebot-website/
├── index.html      # 主页面
├── styles.css      # 样式文件
├── main.js         # JavaScript 交互
├── README.md       # 项目说明
└── LICENSE         # 许可证
```

## 🚀 部署

本项目使用 GitHub Pages 进行自动部署。每次推送到 `main` 分支后，网站将自动更新。
